JavaScript
==========

JavaScript Design Patterns

设计模式类别：

1. 创建型设计模式
   1. Constructor (构造器)
   2. Factory (工厂)
   3. Abstract (抽象)
   4. Prototype (原型)
   5. Singleton (单例/单体)
   6. Builder (生产器)

2. 结构型设计模式
   1. Decorator (装饰者)
   2. Facade (外观)
   3. Flyweight (享元)
   4. Adapter (适配器)
   5. Proxy (代理)

3. 行为设计模式
   1. Iterator (迭代器)
   2. Mediator (中介者)
   3. Observer (观察者)
   4. Visitor (访问者)
